Code Review - Social Network

What is Code Review ?

Code Review is a place where programers can publish snippets of their code so other developers can see it and help them. Much like a social network, the users can publish their status and their likes, so they can be connect to alike minds.

Why build it ?

Although it is a big project, I started, mainly, because as a self-taught programer sometimes can be difficult to know the code you wrote is good or how can it be improved. Sometimes the journey can be harsh and lonely, thats another reason I started this project. Also, I wanted to create a fullstack application to understend how the back-end connects to the front-end in big scale project.
